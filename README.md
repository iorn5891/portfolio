### 全てのファイルは、create-react-appで作成された環境で動作します
* 環境作成方法(※windowsの場合)

  1. コマンドで、"npx create-react-app 〇〇〇(プロジェクト名)"を実行  
  2. 作成したプロジェクトのsrcとpublicを入れ替える  
  3. コマンドで作成したプロジェクトにcdで移動し、npm startを実行するとアプリが起動する
